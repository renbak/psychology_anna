Sapid Website Introduction

The Sapid health theme is developed for making appointments, providing support, writing blogs, etc. You can use this template for your hospital, dental clinic, pharmacy and various other health-related sectors. We developed this template using the latest web technologies like SCSS and Bootstrap 5.

Besides, this healthcare template includes features such as superfast loading, valid HTML 5 & CSS 3, based on Bootstrap 5, 100% responsiveness, easy customizations, smooth page transitions, animations, effects, and many more features. We hope that this theme would come in real handy for a quick website deployment!

Sass Installation Guides

Step 1- Install Node Js

The first need is that you have to install Node js from https://nodejs.org/en/download/ on your PC.

Step 2- Install Ruby

The second need is you have to install Ruby from https://rubyinstaller.org/.

Step 3-Put Command line in Command Prompt

npm install -g sass

Step 4-For Start Sass

watch scss:css

Step 5- In Order to review the changes made in the SCSS you need to run the below command

Npm run build

Variables

Consider variables to be a place to keep information that you want to reuse throughout your CSS. Colors, font stacks, and any CSS value you believe you'll want to repeat can be saved.
assets/scss/variable.scss



Webpack Installation Guides

Getting up and Running

The first need is that you must have Node js and npm installed on your PC. If you haven’t got Node yet, you can either download it from the Node website

First, you go to the Project folder and then open the command prompt For example- look at the command below.

cd Sapid-HTML

npm install

This will install jQuery, plus Slick Slider and Lightbox2 — two plugins we’ll be using on the site — to a node_modules folder at the root of the project.

Note: In Order to make changes in the SCSS and Script, you need to run the below command

Npm run build
